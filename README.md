## ⚙️ Running Locally

To run this portfolio on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MouadDk/Portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Portfolio
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
